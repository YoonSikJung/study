### 계수정렬

O(n)의 시간 복잡도를 보여준다.

하지만 단점 혹은 제한사항이 존재한다.

가장 큰 문제점은 메모리 낭비가 심할 수 있다는 것이다.

만일 정렬을 하려는 원소들 중 10억이란 수가 존재한다면 정렬을 위한 배열(리스트)의 개수가 10억이 되어야 한다.

그래서 계수 정렬같은 경우 정렬하려는 인자의 범위가 비교적 적은 경우 힘을 발휘 할 수 있다.

```python
val = [1,4,3]

count = [0] * (max(val) + 1)

for i in val :
    count[i] +=1

for i, val in enumerate(count) :
    for j in range(val) :
        print(i, end=' ')
```
